Project Overview

When I first opened Soup Surreal’s website, there were many things/elements that caught my eye that I believe ruined its visual design and aesthetic. The first and most prominent element is the website's lack of visual hierarchy. On the landing page, there are essentially no really big main headers or images to catch the eyes of the viewers. It is confusing where to focus your attention because all of the elements on the page are of the same size fighting for attention. The images for instance are both the same size, while the three text statements remain the same size as well making it difficult to decide which element to focus on.

Redesign

Soup Surreal’s website re-design changes the website's visuals, usability, and content. Most of the changes to the website were done through the visual elements, where hierarchy, negative space, colors, and typography were heavily considered. In the original website, order and negative space were non-existent, which made it difficult for users to digest information. The re-design fixes these problems by adding headings and font sizes for hierarchy while providing a cleaner-looking layout that adds negative space between elements. This is exemplified in the menu screen, as I created a card-like element that is spaced out evenly throughout the screen. Hierarchy is displayed through the big headers.
